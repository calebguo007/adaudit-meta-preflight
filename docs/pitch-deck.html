Six assertions the LLM cannot bypass.

Program-level checks comparing fields in the LLM's output against hard limits. Not LLM self-checks. Code overrides the agent.

01 PASS
budget_ad_set_limit_applied
Ad-set count is capped by budget size. $500 cannot fragment into more than 2 ad sets, period.
02 PASS
delivery_objective_applied
Selected objective must match the recommended campaign type. No silent override.
03 PASS
objective_pixel_safety
Conversion objective requires verified pixel events. Cold pixel forces fallback to leads.
04 PASS
economics_safety
Target CPA must be mathematically achievable given AOV, gross margin, CPC band.
05 PASS
risky_claim_rewritten
Outcome-promise and time-bound guarantee claims are replaced with proof-based copy.
06 PASS
timeline_order
Learning-phase budget allocation precedes any scaling. Order is enforced, not suggested.
LLM proposes. Code enforces.
